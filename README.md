# Nomad Fits 👕✈️

Nomad Fits is a modern e-commerce fashion website designed for travelers and style-conscious individuals. The project provides a seamless shopping experience with product browsing, filtering, sorting, cart management, and product details.

## 🚀 Features

- Browse fashion products from an external API
- Product category filtering
- Product sorting (Price, Best Sellers, New Arrivals)
- Product details modal
- Shopping cart functionality
- Add, remove, and update cart items
- Cart data saved using Local Storage
- Responsive design for desktop and mobile
- Contact form with validation
- Customer reviews section
- Toast notifications for user actions

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Fake Store Product API
